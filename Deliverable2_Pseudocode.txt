1. Correct selection of algorithms for key methods

Method: addGame(game)
1. Receive a Game object as a parameter.
2. Add the Game object to the ArrayList of games.
3. End the method.

Method: getNumberOfGames()
1. Check the size of the ArrayList.
2. Return the number of games stored in the ArrayList.

Method: findGameByTitle(title)
1. Receive the String title as a parameter.
2. Loop through each Game object in the ArrayList.
3. Compare the current game's title with the title parameter.
4. If the titles match, return that Game object.
5. If the loop finishes with no match, return null.

Method: removeGame(title)
1. Receive a String title as a parameter.
2. Search for the game using findGameByTitle(title).
3. If the game is found, remove it from the ArrayList.
4. Return true to show the game was removed.
5. If the game is not found, return false.

Method: getAverageRating()
1. If the ArrayList is empty, return 0.0.
2. Create a double variable called total and set it to 0.0.
3. Loop through every Game object in the ArrayList.
4. Add each game's rating to total.
5. Divide total by the number of games in the ArrayList.
6. Return the average rating.

2. Correct selection of primitive data types and Java standard library

String title: stores the game's title.
Genre genre: enum that stores the game's genre.
Platform platform: enum that stores the game's platform.
double rating: stores a decimal rating.
int playTime: stores number of hours played.
boolean internetRequired: stores true or false for online games.
ArrayList<Game> games: Java standard library class used to store many Game objects.
