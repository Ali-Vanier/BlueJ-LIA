PROJECT TITLE:
Game Library Manager

PURPOSE OF PROJECT:
This project is a text-based game library application. It lets users manage a collection of video games. You can add games, rate them, filter them by genre or platform and track how long you played them.

VERSION or DATE:
May 2026

HOW TO START THIS PROJECT:
1. Open BlueJ
2. Open the project folder called BlueJ-LIA
3. Click Compile
4. Right click on the Main class
5. Click void main(String[] args)
6. Click OK
7. The menu will appear in the terminal window

AUTHORS:
Muhammad Ali Babar & Julius Lightburn

USER INSTRUCTIONS:
When the program starts you will see a menu with options numbered 0 to 9.
Type the number of the option you want and press Enter.

1 - Shows how many games are in the library and the average rating
2 - Lists all games with their details
3 - Filter games by genre or platform
4 - Shows the top 3 highest rated games
5 - Shows full details of a specific game
6 - Rate a game out of 5
7 - Log a game session and track play time
8 - See all your past game sessions
9 - Add your own game to the library
0 - Exit the program
